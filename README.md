نظام الاستمارات الذكي — نقش Judiyya
منصة ويب متكاملة لبناء الاستمارات، جمع البيانات، وتحليل النتائج مع نظام تقييم ذكي ومخفي.

✨ المميزات الأساسية
نظام أدوار متكامل: مدير (Admin)، مستشار (Counselor)، ومستخدم عادي (Responder).
بناء استمارات مرن: يدعم نصوص، أرقام، تواريخ، اختيار واحد/متعدد، مقاييس خطية، ومصفوفات.
نظام تقييم مخفي: تخصيص نقاط لكل إجابة وتحديد مستويات تقييم (مخفية عن المستخدم تظهر له فقط كتقييم نهائي).
لوحات تحليلات تفاعلية: رسوم بيانية (Chart.js) للمقارنات والإحصائيات.
مشاركة النتائج بطاقات: إنشاء بطاقة نتيجة أنيقة وتحميلها كصورة (html2canvas).
مصادقة متعددة: تسجيل دخول عبر جوجل، بريد إلكتروني/كلمة مرور، أو كزائر (Guest).
🛠️ التقنيات المستخدمة
Frontend: HTML5, CSS3, Vanilla JavaScript
Styling: Tailwind CSS, Cairo Font, FontAwesome Icons
Backend & Database: Firebase (Authentication, Cloud Firestore)
Charts: Chart.js
Image Capture: html2canvas
